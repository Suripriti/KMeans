# Movie Recommendation System using KMeans Clustering

This project is a Movie Recommendation System built using:

- Streamlit
- KMeans Clustering
- Scikit-learn
- TMDB Movie Dataset

The application recommends similar movies based on movie clusters.

---

TMDB 5000 Movie Dataset
